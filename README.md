# 🚀 HAN with QKV Attention for IoT Intrusion Detection

## 📌 Project Description
This project implements a **Hierarchical Attention Network (HAN)** enhanced with a **Query-Key-Value (QKV) attention mechanism** for detecting cyber-attacks in IoT environments.  

The model is trained and evaluated using:
- BoT-IoT Dataset  
- TON-IoT Dataset  

It aims to achieve high accuracy and robust performance in identifying malicious network traffic.

---

## 🎯 Objectives
- Build an efficient intrusion detection system  
- Enhance feature learning using QKV attention  
- Achieve high accuracy (>98%)  
- Compare performance across multiple IoT datasets  

---

## 🧠 Model Architecture
The system consists of the following components:

1. Input Layer – Preprocessed network traffic data  
2. Embedding Layer – Feature representation  
3. Hierarchical Attention Network (HAN)  
4. QKV Attention Mechanism  
5. Fully Connected Dense Layers  
6. Output Layer (Binary Classification)
